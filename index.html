What we should expect from tests?

Confidence in what we code, both because it works and because it does not break previously working code.

What we should expect from tests?

  • We cannot totally rely on manual testing because
  • We will miss the whole picture
  • It takes a lot of time to do it properly
  • It does not scale
  • It includes a falible component (we)
